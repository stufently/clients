Regression Testing Areas

1. Ctrl+Shift+9 — With vault unlocked, confirm shortcut generates a password to clipboard and adds it to history.
2. Right-click context menu (Copy Password) — Right-click a page, choose "Copy Password"; confirm a password is copied to the clipboard and appears in generator history.
3. Inline menu (password field, first open) — Focus a password field on a registration form; confirm a generated password appears immediately (not blank).
4. Inline menu (refresh) — Click the refresh icon; confirm the displayed password changes to a new value.
5. Inline menu (fill) — Click "use this password"; confirm it fills the field, the filled value matches what was shown, and the save login prompt appears.
6. Inline menu (clear on navigation) — Generate a password, navigate to another page or switch tabs and return; confirm the old password is not shown on re-focus.
7. Account creation (tab between fields) — On a registration form, tab username → password → username → password; confirm the generator view updates correctly each time without showing a stale or blank password.
8. Save login prompt — Fill a generated password into a registration form; confirm the inline menu transitions cleanly to the save login view with no blank state in between.
