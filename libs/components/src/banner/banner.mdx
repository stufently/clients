## Content Slots

Banners support three content projection slots.

### Default slot

Plain text or inline elements. Use `bitLink` with `[linkType]` matching the banner `variant` for
inline links.

### Title slot (`slot="title"`)

Renders a heading above the body text. When a title is projected, the actions slot becomes visible
and the layout shifts to its expanded form. Apply `slot="title"` to any element.

### Actions slot (`slot="actions"`)

Only visible when a title is projected. At wide widths the actions are right-aligned; at narrow
widths they appear below the body text. Wrap action elements in `ng-container slot="actions"`.

## Accessibility

Banners set the `role="status"` and `aria-live="polite"` attributes to ensure screen readers
announce the content prior to the rest of the page. This behavior can be disabled by setting
`[useAlertRole]="false"`.
